## 1. 클로드 코드와 에이전틱 코딩 개요
단순한 코드 자동완성을 넘어, AI가 스스로 코드를 분석, 계획, 작성, 테스트까지 수행하는 '에이전틱 코딩(Agentic Coding)'을 터미널에서 구현합니다.

- **에이전틱 루프**: `컨텍스트 수집` ➔ `계획 수립 및 작업 수행` ➔ `결과 확인`의 3단계 순환 구조로 작동하며, 작성한 코드를 스스로 테스트하고 에러를 검증해 수정합니다.
- **탐색-계획-구현-커밋(Explore-Plan-Implement-Commit)**: 앤트로픽이 정의한 핵심 워크플로로, 코드의 실제 수정(Write, Edit) 전 안전한 샌드박스 환경과 사용자 승인 절차를 거치게 됩니다.

## 2. 컨텍스트 및 접근 영역
로컬 개발 환경의 시스템과 파일에 직접 접근하여 작업의 전체 맥락을 파악합니다.

- **5대 접근 영역**:
  1. **파일 시스템**: Read, Write, Edit, Glob, Grep 도구 활용 (수정 시 사용자 승인 필요).
  2. **터미널 (Bash)**: npm, git, 빌드 스크립트 등 로컬 명령어를 직접 실행.
  3. **웹**: WebSearch, WebFetch를 통해 인터넷의 최신 문서 및 기술 블로그 검색.
  4. **MCP 서버**: 외부 서비스(GitHub, DB, Sentry, Figma 등) 데이터 연동.
  5. **에디터 컨텍스트**: VS Code, JetBrains 등 IDE의 파일 상태 및 린트 진단 정보 연동.
- **컨텍스트 윈도우 관리**: 최대 20만 토큰을 지원하며, 낭비를 막기 위해 압축(Compaction), 점진적 공개(Progressive disclosure), 서브에이전트 위임, 구조화된 메모 작성 등의 최적화 전략을 사용합니다.

## 3. 설치 및 인증
macOS 13 이상, 리눅스(우분투 20.04+), Windows(WSL2 환경)를 지원합니다.

- **기본 설치 (macOS/리눅스)**:
  ```bash
  curl -fsSL https://claude.ai/install.sh | bash
  ```
  ```
  *(네이티브 설치 시 백그라운드 자동 업데이트 지원)*
- **인증 방식**: Pro/Max 요금제 가입자의 경우 `claude.ai` 계정 로그인을 사용하며, 팀 단위 과금이 필요한 경우 클로드 콘솔(OAuth)이나 Bedrock, Vertex AI 같은 클라우드 플랫폼 연동을 활용합니다.

## 4. 핵심 CLI 명령어 및 기능
일반적인 대화형 모드와 쉘 스크립트 연동을 위한 비대화형(Headless) 모드를 모두 지원합니다.

- **주요 실행 플래그**:
  - `claude`: 대화형 세션 시작
  - `claude -p "질문"`: 비대화형 쿼리 실행 후 즉시 종료
  - `claude -c` (또는 `-r`): 이전 세션 이어서 작업 재개
  - `claude doctor`: 설치 상태 및 환경 문제 진단
- **세션 내 명령어**: `/help`(도움말), `/clear`(대화 컨텍스트 초기화) 등.
- **체크포인트 & 되감기(Rewind)**: 프롬프트 입력 시점마다 파일 상태를 자동 체크포인트로 저장합니다. 잘못된 수정이 발생하면 `/rewind` 명령어 또는 `Esc` 키 2번 입력으로 코드와 대화 내역을 즉시 롤백할 수 있습니다.

## 5. 작업 파이프라인 및 Git 연동
특정 프로젝트 디렉토리에서 실행하면 기존 `.git` 히스토리와 설정(`CLAUDE.md`)을 자동 스캔하여 컨텍스트를 구성합니다.

- **Git 통합 워크플로**: `claude commit` 명령을 활용하거나 자연어로 "변경된 파일 스테이징하고 커밋해줘"라고 입력하면, 현재 상태(`git status`, `git diff`)를 스스로 파악하여 적절한 커밋 메시지를 자동 작성하고 브랜치를 생성합니다.
- **명령어 파이프 연동**: 기존 유닉스 도구와 결합 가능. 파이프(`|`)를 통해 외부 데이터를 클로드 코드의 입력으로 직접 넘길 수 있습니다 (예: 대량 로그 분석 자동화).
```
